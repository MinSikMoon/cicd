# cicd
cicd 정리

#### 머리말
- 현재 맡고 있는 시스템의 빌드, 배포 자동화를 구축하기 위해서, CI/CD와 관련된 지식들을 공부해놓고 정리해놓으려 한다.
- 자동화 테스트는, 현재 테스트 주도 방식 개발이 몸에 익으면 서서히 도입해 넣을 예정이다.

#### 현재 상황
- 패키징, 정적분석, 배포, 톰캣 재시작 까지
